## Змінні та константи
Для зберігання даних у програмі використовуються змінні. Змінні призначені для зберігання будь-яких тимчасових даних або даних, які в процесі роботи можуть змінювати своє значення.

### Оголошення змінних та конствнт
Для створення змінних застосовуються зарезервовані слова:
- `var` - створення змінної
- `let` - створення змінної
- `const` - створення константи

Спочатку можна було створювати змінні лише використовуючи `var`, але цей метод створення призводив до спливання значень у батьківські лексичні області(це буде розглянуто в одній із наступних лекцій).
Але з появою специфікації ES2015 (ES6), були додані `let` і `const`.
Наразі `краще` використовувати `let` і `const`, це позбавить ваш код зайвих непередбачених помилок.

Оголошення змінної - це окрема інструкція, тому завершується крапкою з комою.

```javascript
var firstName;
let lastName;
const someConstantValue;

// Можна створювати змінні через кому
let username, age, heigt;
let a, b, c;

// Це дві різні змінні
let username;
let userName;
```

__ВАЖЛИВО: Кожна окрема інструкція завершуеться крапкою з комою__
 
Якщо ви впевнені, що значення в процесі програми не змінюватиметься, тоді це значення визначається як константа. Якщо невідомо, чи змінюватиметься чи ні, то рекомендується визначити значення як константу. А якщо в подальшому потрібно його змінити, то легко можна змінити визначення значення з `const` на `let`.


Кожна змінна має ім'я. Ім'я є довільним набором алфавітно-цифрових символів, символу підкреслення `_` або символу долара `$`, причому назви не повинні починатися з цифрових символів. Тобто ми можемо використовувати у назві букви, цифри, підкреслення. Однак усі інші символи заборонені.

### Привласнення значення змінній

Після створення змінної їй можна надати якесь значення. Процес надання змінної початкового значення називається `ініціалізацією`. Для цього застосовується оператор присвоєння `=`:

```javascript
let userName;
userName = "Alex";
```

Можна одразу надати змінній значення при її визначенні:
```javascript
let userName = "Alex";
let userAge = 37;
```

## Типи даних

Всі використовувані дані в JavaScript мають певний тип.

Типи даних JavaScript:

| Тип даних | Опис |
|-----------|------|
| `String`  | `рядок` символів |
| `Number`  | `числове` значення |
| `BigInt`  | `числове` значення, зі збільшеним діапазоном чисел |
| `Boolean` | `логічне` значення, `true` або `false` |
| `Undefined`| `пусте` значення, яке набуває не оголошена, але не ініціалізована змінна |
| `Null`    | `пусте` значення, яке може привласнити змінній користувач |
| `Symbol`  | `унікальнє` значенням, яке часто застосовується для звернення до властивостей складних об'єктів |
| `Object`  | `комплексний об'єкт`, що складається зі значень інших типів чи інших об'єктів |
___


### `String`

Тип `String` представляє символьні рядки. Для створення рядків застосовуються:
 - подвійні лапки
 - одинарні лапки
 - косі лапки
  
  Єдине обмеження: тип лапки, що закриває, повинен бути той же, що і тип відкриває, тобто або обидві подвійні, або обидві одинарні.

```javascript
let user = "Tom";
let company = 'Microsoft';
let language = `JavaScript`;
```

Якщо всередині рядка трапляються лапки, то ми їх повинні екранувати слішем або використовувати інший тип лапок.

```javascript
let company = "Копманія \"Амазон\""; // екранування
let company1 = "Копманія 'Амазон'"; // різні лапки
let company2 = 'Копманія "Амазон"'; // різні лапки
```
#### Інтерполяція
Використання косих лапок дозволяє нам застосовувати такий прийом як `інтерполяція` - вбудовувати дані в рядок.
```javascript
let user = "Tom";
let age = 37;
let isMarried = false;

let text = `Name: ${user}`; // інтерполяція
let text = `Name: ${user}   Age: ${age}   IsMarried: ${isMarried}`; // інтерполяція
```

### `Number`

Тип Number представляє числа, які можуть бути:
 - `цілими`
 - `дробовими`

Вони можуть набувати `позитивних`, і `негативних` значень. 

```javascript
let integer = 45;
let float = 23.897;
```
Як роздільник між цілою та дробовою частинами, як і в інших мовах програмування, використовується `крапка`.


### `BigInt`

Тип `BigInt` був доданий в останніх стандартах JavaScript. Він використовується для представлення дуже великих цілих чисел, що виходять за межі діапазону типу `Number`.

`Number` не може забезпечити роботу з числами за межами свого діапазону:

```javascript
let num = 9007199254740991
console.log(num);       // 9007199254740991
console.log(num + 1);   // 9007199254740992
console.log(num + 2);   // 9007199254740992
```

Для створення числа з типом `BigInt` наприкінці числа додається суфікс `n`:

```javascript
let dimension = 19007n;
const value = 2545n;
let myBigInt = 9007199254740991n;
console.log(dimension); // 19007n
console.log(value); // 2545n
console.log(myBigInt + 1n);   // 9007199254740992n
console.log(myBigInt + 2n);   // 9007199254740993n
console.log(myBigInt + 10n);  // 9007199254741001n
```


### `Boolean`

Тип `Boolean` являє собою булеві або логічні значення `true` (вірно) і `false` (хибно):

```javascript
let done = true;
let isAvailable = false;
```


### `null` та `undefined`

`undefined` вказує, що значення не визначено. Наприклад, коли ми тільки оголосили змінну без надання їй початкового значення, вона представляє тип `undefined`:
```javascript
let isAlive;
console.log(isAlive); // undefined
```

Присвоєння значення `null` означає, що змінна відсутня значення:
```javascript
let isAlive;
console.log(isAlive); // undefined
isAlive = null;
console.log(isAlive); // null
isAlive = undefined;
console.log(isAlive); // undefined
```


### `Object`

Тип `Object` - складний об'єкт. Оголосити об'єкт можна за допомогою фігурних дужок:
```javascript
let student = {};
```

Об'єкт може мати різні `властивості` та `методи`.

__Властивості - це змінні, а методи це функції.__

__Доступ до властивостей і методів надає оператор доступу `.` (крапка)__
```javascript
const student = {
  firstName: "Ivan",
  lastName: "Kovalenko",
  age: 20,
  spec: "Applied Linguistics",
  course: 3,
};
console.log(student.firstName); // Ivan
console.log(student.lastName);  // Kovalenko
console.log(student.age);       // 20
console.log(student.spec);      // Applied Linguistics
console.log(student.course);    // 3
```

Детальніше об'єкти розглянемо пізніше.


### `Слабка та динамічна типізація`

JavaScript є мовою зі `слабкою` та `динамічною` типізацією. Це означає, що змінні можуть динамічно змінювати тип.

```javascript
let sampleVariable;  // тип undefined
sampleVariable = 45;  // тип number
sampleVariable = "45"; // тип string
```
Незважаючи на те, що в другому і третьому випадку консоль виведе нам число 45, але в другому змінна id буде представляти число, а в третьому випадку - рядок.


### Оператор `typeof`

За допомогою оператора typeof можна отримати тип змінної:

```javascript
let id;
console.log(typeof id); // undefined
 
id = 45;
console.log(typeof id); // number
 
id = 45n;
console.log(typeof id); // bigint
 
id = "45";
console.log(typeof id); // string

console.log(typeof null); // object
```

__Варто зазначити, що для значення `null` оператор `typeof` повертає значення `"object"`, незважаючи на те, що відповідно до специфікації JavaScript значення null є окремим типом__

## Математичні оператори

JavaScript підтримує всі базові математичні операції:
| Оператор | Операція |
|----------|----------|
|   `+`   | додавання|
|   `-`   | віднімання |
|   `*`   | множення |
|   `/`   | ділення |
|   `%`   | отримання залишку від ділення |
|   `**`   | зведення в ступінь |
|   `++`   | інкрементація числа (n+1) |
|   `--`   | декриментація числа (n-1) |

```javascript
let x = 10 + 50; // додавання
let y = 100 - 30; // віднімання
let mult = x * y; // множення
let div = x / y; // ділення
let rem = 10 % 3; // залишок від ділення
let pow1 = 2 ** 3; // зведення в ступінь
let pow2 = x ** 2; // зведення в ступінь
```

### Інкремент та декримент

Оператор інкременту `++` збільшує змінну на одиницю, оператор декременту `--` зменшує значення змінної на одиницю. Існують `префіксний` та `постфікстний` інкременти.

`Префіксний` - спочатку збільшує змінну одиницю, а потім повертає її значення.

`Постфіксний` - спочатку повертає значення змінної, а потім збільшує його на одиницю.

Постфіксний інкремент
```javascript
let num1 = 10;
const preInc = ++num1;
console.log(preInc);
```
Постфіксний інкремент
```javascript
let num2 = 10;
const postInc = num2++;
console.log(postInc);
```

рефіксний декремент
```javascript
let num3 = 10;
const preDec = --num1;
console.log(preDec);
```
постфіксний декремент
```javascript
let num4 = 10;
const postDec = num2--;
console.log(postDec);
```

### Виконання математичних операцій

Так само, як і в математиці, всі операції:
1. Виконуються зліва направо та розрізняються за пріоритетами;
2. Виконуються операції інкременту та декременту;
3. Виконуються операції множення та ділення
4. Виконуються операції додавання та віднімання.

__Щоб змінити стандартний хід виконання операцій, частину виразів можна помістити у дужки__

```javascript
let ten = 10;
let mathResult = 5 + (6 - 2) * --ten;
console.log(mathResult); // 41
```

## Оператори порівняння

| Оператор | Операція |
|----------|----------|
| `==`  | перевірка рівності |
| `===` | сувора перевірка рівності |
| `!=`  | перевірка `НЕ`рівності |
| `!==` | сувора перевірка `НЕ`рівності |
| `>`   | порівняння більшого з меншим значенням |
| `<`   | порівняння меншого з більшим значенням |
| `>=`  | порівняння більшості або рівності |
| `<=`  | порівняння меншості або рівності |

Якщо рівність вірна, то вираз повертає `true`, а якщо ні - `false`.

__Будьте обережні з операторами `==` та `!=` у `JavaScript`. Вони можуть повертати `true`, якщо значення різного типу.__

```javascript
console.log(123 == 123); // true
console.log(123 == "123"); // true

console.log(10 === 10); // true
console.log(10 === "10"); // false

console.log("word" != "word"); // false
console.log("300" != 300); // false

console.log("word" !== "word"); // false
console.log("300" !== 300); // true

console.log(12 > 9); // true
console.log(12 < 9); // false

console.log(300 > 450); // false
console.log(300 < 500); // true

console.log(1 <= 0); // false
console.log(1 <= 1); // true

console.log(20 >= 24); // false
console.log(320 >= 250); // true
```

## Логічні оператори

Логічні операції застосовуються для об'єднання результатів двох операцій порівняння, або виразів, результатом котрих буде значення `Boolean`. Тобто порівнює між собою значення `true` та `false`.

| Оператор | Операція |
|----------|----------|
| `&&`  | порівнює два вирази на правдивість |
| <code>&#124;&#124;</code> | порівнює два вирази на правдивість |
| `!`   | привласнює виразу протележне значення |

| Оператор | Логічне |
|----------|----------|
| `&&`  | AND (І) |
| <code>&#124;&#124;</code> | OR (АБО) |
| `!`   | NOT (НЕ) |

```javascript
console.log(false && false);  // false
console.log(true && false);   // false
console.log(false && true);   // false
console.log(true && true);    // true

console.log(false || false);  // false
console.log(true || false);   // true
console.log(false || true);   // true
console.log(true || true);    // true

console.log(!true);           // false
console.log(!false);          // true
```

## Оператори привласнення

За допомогою операторів привласнення можна записувати значення у змінну. Також можна змінювання під час запису, якщо об'єднати математичний оператор з оператором `=`.

| Оператор | Операція |
|----------|----------|
| `=`   | просте привласнення значння |
| `+=`  | додавання значення до попереднього |
| `-=`  | віднімання значення від попереднього |
| `*=`  | множення значння з переднім |
| `/=`  | ділення попереднього значння |
| `**=` | зведення в ступінь значення |
| `%=`  | отримання залишку ділення |
| `&&=` | привласнює `true`, якщо обидва значчення `true` |
| <code>&#124;&#124;=</code>   | привласнює `true`, якщо одне із значень `true` |


```javascript
let sum = 23;
sum += 5;
console.log(sum); // 28

let sub = 28;
sub -= 10;
console.log(sub); // 18

let multip = 20;
multip *= 2;
console.log(multip); // 40

let divided = 40;
divided /= 4;
console.log(divided); // 10

let pow = 5;
pow **= 2;
console.log(pow); // 25

let afterFloatDot = 10;
afterFloatDot %= 3;
console.log(afterFloatDot); // 1


let res0 = false;
let bool0 = false;
res0 &&= bool0;
console.log(res0); // false

let res1 = true;
let bool1 = false;
res1 &&= bool1;
console.log(res1); // false
 
let res2 = false;
let bool2 = true;
res2 &&= bool2;
console.log(res2); // false
 
let res3 = true;
let bool3 = true;
res3 &&= bool3;
console.log(res3); // true
 

let res4 = false;
let bool4 = false;
res4 ||= bool4;
console.log(res4); // false

let res5 = true;
let bool5 = false;
res5 ||= bool5;
console.log(res5); // true
 
let res6 = false;
let bool6 = true;
res6 ||= bool6;
console.log(res6); // true
 
let res7 = true;
let bool7 = true;
res7 ||= bool7;
console.log(res7); // true
```
